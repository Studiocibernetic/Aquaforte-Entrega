# Aquaforte - Serviços de Canalização em Lisboa

## Sobre o Projeto

Este é um website profissional para a Aquaforte, especializada em serviços de canalização em Lisboa.

## Como editar este código?

Existem várias formas de editar esta aplicação.

**Usar um IDE local**

Se preferir trabalhar localmente com o seu IDE favorito, pode clonar este repositório e fazer alterações. 

Requisitos:
- Node.js & npm instalados - [instalar com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga estes passos:

```sh
# Passo 1: Clone o repositório usando o URL Git.
git clone <YOUR_GIT_URL>

# Passo 2: Navegue até ao diretório do projeto.
cd <YOUR_PROJECT_NAME>

# Passo 3: Instale as dependências necessárias.
npm i

# Passo 4: Inicie o servidor de desenvolvimento com recarregamento automático.
npm run dev
```

**Editar diretamente no GitHub**

- Navegue até aos ficheiros desejados.
- Clique no botão "Editar" (ícone de lápis) no canto superior direito da visualização do ficheiro.
- Faça as alterações e commite-as.

**Usar GitHub Codespaces**

- Navegue até à página principal do repositório.
- Clique no botão "Código" (botão verde) próximo do canto superior direito.
- Selecione o separador "Codespaces".
- Clique em "Novo codespace" para lançar um novo ambiente Codespace.
- Edite os ficheiros diretamente no Codespace e commite e envie as alterações quando terminar.

## Como posso implantar este projeto?

Pode implantar este projeto usando qualquer serviço de hospedagem moderno como Vercel, Netlify ou GitHub Pages.

## Posso conectar um domínio personalizado?

Sim, pode!

Para conectar um domínio, configure os registos DNS apropriados com o seu provedor de domínio.

Consulte a documentação do seu serviço de hospedagem para obter instruções específicas sobre como configurar um domínio personalizado.
